# Oxidy

Oxidy is a terminal based code editor written in rust. It uses Crossterm to render onto the terminal in an AlternateScreen.

### Features

- [ ] Line wrapping / Line scrolling
- [ ] Undo/Redo
- [ ] Buffer
- [ ] File management
- [ ] Color schemes
- [ ] LSP
    - [ ] syntax highlighting
    - [ ] auto complete
    - [ ] type hints
